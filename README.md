# 🚀 Surman Code Hub - Next-Gen 3D Landing Page

A premium, high-performance, responsive landing page engineered with advanced 3D perspective layering, custom ambient particle physics, and an interactive 3D branding advertisement matrix.

> **💡 Developer Note:** This project is completely open-source and customizable! Feel free to fork, clone, modify, and use this template for your own projects or portfolios.

---

## ✨ Features

*   **Dual-Layer Animation Architecture:** A smooth background engine rendering floating ambient stars simultaneously with foreground actions.
*   **3D Brand Advertising Matrix:** Moving the cursor launches clear, high-fidelity **"SURMAN CODE HUB"** neon 3D typography arrays that rise and scale fluidly.
*   **Executive Left-Wing Section:** Dedicated branding zone displaying corporate taglines, company vision, and the official **Founder & CEO Profile Badge (Vishal Kumar)**.
*   **3D Push-and-Swing Workspace:** Login and Signup cards leverage CSS depth perception metrics (`perspective: 1500px`) to switch via realistic 3D space rotations.
